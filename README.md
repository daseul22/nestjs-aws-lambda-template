## Installation

### AWS Lambda, serverelss framework

```bash
$ npm install -g serverless
```

### Dependencies

```bash
$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# serverless offline
$ npm run sls:offline

# serverless deploy:dev
$ npm run sls:deploy:dev

# serverless deploy:staging
$ npm run sls:deploy:staging

# serverless deploy:prod
$ npm run sls:deploy:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
